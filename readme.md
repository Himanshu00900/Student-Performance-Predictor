# 🎓 Student Performance Predictor

A Machine Learning web application that predicts student final marks using:

- Study Hours
- Attendance Percentage
- Previous Marks

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Streamlit

## Features

- Student Performance Prediction
- Performance Categories
- Recommendations
- Correlation Heatmap
- Prediction History
- Dataset Statistics

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
